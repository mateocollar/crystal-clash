# Crystal Clash

Crystal Clash is a minigame based on "Gem Grab" from Brawl Stars for Craftsman, featuring many advanced functions and mechanics for extensive entertainment.

## Brands

- PHP 7+
- Git
- GitHub

## Links

- Repository: [CrystalClash](https://github.com/mateocollar/crystal-clash)
- Discord: [PocketMine 2.0.0](https://discord.gg/4D5qqVxdK8)
- License: ./LICENSE

## Usage

```Download the file "CrystalClash_v1.0.phar" and place it in your "plugins" folder, Restart the server and it will be active. 
```

## Features

- Cup System.
- Gadgets & Hypercargas.
- Statistics system.

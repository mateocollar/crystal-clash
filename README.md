# Crystal Clash

Crystal Clash is a minigame based on "Gem Grab" from Brawl Stars for Craftsman, featuring many advanced functions and mechanics for extensive entertainment.

## Brands

- PHP 7+
- Git
- GitHub

## Links

## Usage

```php
Profiler::start("database");

// Code...

Profiler::end("database");
Profiler::dumpEntry("database");
```

```php
Profiler::run("cleanArena", function () {
    // Code...
}, $arg1);
    
Profiler::dumpDetailed();
```

## Features

- Lightweight.
- Zero dependencies.
- Callback support.
- Named profiling entries.
- Timing statistics.
- Simple and detailed output.
